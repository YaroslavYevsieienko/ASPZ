# Обчислення багатовимірного масиву

## Функції

* Обчислення загальної кількості елементів багатовимірного масиву
* Перетворення лінійного індексу у координати багатовимірного масиву
* Заповнення масиву випадковими числами з плаваючою точкою
* Вивід елементів масиву
* Пошук усіх індексів заданого числа з урахуванням EPS
* Динамічне виділення та звільнення пам’яті

## Програмні вимоги

* Середовище запуску (консоль Windows CMD, PowerShell або Linux термінал)
* Компілятор GCC
* Компіляція .c файлу у виконуваний файл з лінкуванням математичної бібліотеки

## Компіляція

* Звичайна збірка: `gcc 1.8.c -o multidim_array -lm`
* Збірка з оптимізацією O2: `gcc -O2 1.8.c -o multidim_array_o2 -lm`
* Збірка з оптимізацією O3: `gcc -O3 1.8.c -o multidim_array_o3 -lm`

## Запуск коду

`./multidim_array`
